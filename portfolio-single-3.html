Project Management System

Project Description

Developed a comprehensive project management system enabling developers to contribute to projects and allowing users to invite other developers via email invitations. Implemented user authentication using JWT to ensure secure access and interactions within the application. Leveraged ReactJS for a dynamic and responsive front-end, providing an intuitive user interface. Utilized Spring Boot for the back-end to handle business logic, RESTful APIs, and secure communication with the MySQL database. Designed the database schema in MySQL to manage users, projects, tasks, and invitations efficiently. Focused on code quality and maintainability, adhering to best practices in software development. Razorpay Integration, Developer Contributions, Email Invitations.
